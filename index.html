Compare two PDF files online, free and without uploading anything

Draftline is a free tool for comparing two PDF files online. Drop in the old version and the new one, and it marks every added and deleted word right on the revised page: green where text was inserted, a red strikethrough marker where text was removed. It works with contracts, leases, statements of work, policy documents, or any two drafts of the same PDF.

Most PDF comparison tools upload your files to a server before they can diff them. Draftline does not. It is a PDF diff tool with no upload step at all: both files are read and compared inside your browser using pdf.js, the same engine Firefox uses to display PDFs. Nothing is sent over the network, so you can safely compare an NDA, a signed lease, or a term sheet you would never paste into a website.

The result reads like a redline a careful reviewer would mark by hand. Instead of dumping plain text into two columns, Draftline paints the diff over the real rendered page, so a reworded clause, a changed dollar amount, or a renumbered section shows up in its actual layout. There is no page limit and no account, so a sixty-page lease compares as easily as a one-page letter.

Frequently asked questions

How do I compare two PDF files online for free?
Open Draftline, drop or choose the two PDF versions you want to compare, and it highlights every change on the page. It is free, there is no sign-up, and there is no limit on how many documents you compare.
Is there a PDF diff tool that works with no upload?
Yes. Draftline compares both PDFs entirely in your browser with pdf.js, so neither file is uploaded to a server. Once the page has loaded you can even go offline and the comparison still works.
Can I redline and compare two contract PDFs?
That is what Draftline is built for. It runs a word-level diff, so a single edited term like a changed rent amount or deadline is marked precisely instead of flagging the whole paragraph, and the marks sit over the original contract layout.
Is there a page limit?
No. Because there is no server doing the work, there is nothing to ration. Long documents stream in one page at a time with a Cancel button, so the tab stays responsive.
What happens with scanned PDFs?
Draftline compares the text layer of each PDF. A scanned or image-only page has no text layer, so it is shown with a note rather than a silent empty comparison. For a meaningful diff, both files need selectable text.
